# 🎓 University Student Manager

Веб-приложение на Flask для управления студентами учебной группы. Поддерживает три роли: **Студент**, **Староста** и **Профорг** — каждая с собственными полями и логикой.

---

## 🛠 Технологии

- **Python 3.10+**
- **Flask** — веб-фреймворк
- **SQLite** — база данных
- **Jinja2** — шаблонизатор

---

## 📁 Структура проекта

```
university/
├── __init__.py          # Инициализация Flask-приложения
├── routes.py            # Маршруты (Blueprint)
├── students.py          # Основная логика (Students)
├── classes.py           # Модели: Student, Starosta, Proforg
├── storage.py           # Работа с БД (DBStorage)
├── flask_io.py          # Ввод/вывод через Flask (FlaskInputOutput)
└── templates/
    ├── form.tpl         # Форма добавления/редактирования
    └── list.tpl         # Список всех записей
```

---

## 🚀 Запуск

```bash
# Установить зависимости
pip install -r requirements.txt
# Запустить приложение
python run_server.py
```

Приложение будет доступно по адресу: [http://localhost:5000](http://localhost:5000)
---

## ⚙️ Функциональность

| Действие | Маршрут | Метод |
|---|---|---|
| Список студентов | `/` | GET |
| Форма добавления/редактирования | `/form/<id>` | GET |
| Сохранить студента | `/add` | POST |
| Удалить студента | `/delete/<id>` | POST |

---

## 👥 Роли

### Студент
Базовая роль. Поля: имя, фамилия, возраст, номер паспорта, группа.

### Староста
Расширяет Студента. Дополнительные поля: дата начала полномочий, контактный номер.

### Профорг
Расширяет Студента. Дополнительные поля: количество организованных мероприятий, оценка эффективности.

---

## 🗄 База данных

SQLite-файл создаётся автоматически при первом запуске по пути `data/book.sqlite`.

Схема таблицы `book`:

```sql
CREATE TABLE book (
    id               INTEGER PRIMARY KEY AUTOINCREMENT,
    passport_number  TEXT,
    name             TEXT,
    surname          TEXT,
    age              INTEGER,
    group_name       TEXT,
    time             TIMESTAMP,
    status_in_group  TEXT,
    additional_data  BLOB  -- доп. поля Старосты и Профорга
);
```
